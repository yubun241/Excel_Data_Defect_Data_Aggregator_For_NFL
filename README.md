## Excel_Data_Defect_Data_Aggregator_For_NFL
Excelファイルから欠点関連データを自動抽出・集計し、統合レポートを生成するPythonツールです。   

複数のExcelファイルを一括処理し、欠点の種類別カウントとヘッダー情報を効率的にまとめます。


## 機能

- ✅ **複数Excelファイルの一括処理** - `../data` フォルダ内の全.xlsxファイルを自動検出
  
- ✅ **シート抽出フィルタリング** - 指定キーワードを含むシートのみを対象化
  
- ✅ **欠点データの自動集計** - 略号別のカウント集計
  
- ✅ **定型項目の抽出** - 生産番号、品質番号など重要情報を自動抽出
  
- ✅ **統合レポート生成** - 複数ファイルのデータを1つのExcelに統合
  
- ✅ **重複排除** - 同一データの重複を自動削除

- ✅ **エラーハンドリング** - 処理エラーを記録して継続実行





## 必要な環境

- Python 3.7以上
  
- pandas
  
- openpyxl

├── result/            # 出力ファイル置き場（自動作成）

## 処理の流れ
1. ../data フォルダから *.xlsx ファイルを検出
   
3. 各ファイルから対象シート（"欠点巻込連絡票"含み、"原本"除外）を抽出
   
5. 指定行・列を読み込み、カラム名を統一
  
7. 略号別に欠点をカウント集計
  
9. 生産番号などのヘッダー情報を抽出
   
10. 集計結果 + ヘッダー情報を結合

11. 全データを統合して重複排除
  
12. 結果をExcelで出力


## 出力ファイル形式

統合されたExcelファイルには以下の列が含まれます：

列名	説明

生産番号	製品の生産番号

品質番号	品質管理番号

工程番号	工程識別番号

トータル長さ	全長

欠点合計長さ	欠点部分の合計長さ

粗化欠け	粗化欠けの欠点数

誤検出	誤検出の欠点数

黒色汚れ	黒色汚れの欠点数

スリップ疵	スリップ疵の欠点数

その他	上記以外の欠点数

プログラム起動日時	処理実行日時
